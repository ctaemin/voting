# 온라인 투표 앱 기획서

## 개요

회사 아이디어 평가 세션에서 참석자들이 QR코드를 통해 스마트폰으로 접속하여 투표하고, 실시간으로 결과를 확인할 수 있는 웹 기반 투표 시스템.

---

## 현재 문제점

- 아이디어 평가 결과를 수작업으로 집계하여 시간 소요 및 오류 가능성 존재
- 실시간 결과 확인 불가

---

## 목표

- 참석자가 QR코드 스캔 → 스마트폰으로 투표 → 결과 즉시 확인
- 5G 환경(모바일 데이터)에서도 접속 가능 (인터넷 배포 필요)

---

## 화면 구성

### 1. 관리자 화면 (`admin.html`)

| 기능 | 설명 |
|------|------|
| 아이디어 등록 | 투표 항목(아이디어) 추가/삭제 |
| 투표 설정 | 1인당 선택 가능 수 설정 (기본: 2개) |
| 투표 시작/종료 | 투표 활성화 제어 |
| QR코드 생성 | 투표 참여 URL을 QR코드로 표시 |
| 실시간 결과 | 실시간 득표 현황 표시 |
| 결과 초기화 | 투표 데이터 리셋 |

### 2. 투표 화면 (`vote.html`)

| 기능 | 설명 |
|------|------|
| 아이디어 목록 | 등록된 아이디어 카드 형식 표시 |
| 선택 UI | 체크박스 방식, 최대 N개 선택 제한 |
| 제출 버튼 | 투표 완료 후 중복 투표 방지 |
| 완료 메시지 | 제출 후 감사 메시지 표시 |

### 3. 결과 화면 (`result.html`)

| 기능 | 설명 |
|------|------|
| 실시간 결과 | 득표 수 기준 정렬된 순위 표시 |
| 진행 바 | 각 아이디어별 득표율 시각화 |
| 자동 갱신 | 일정 주기로 결과 자동 새로고침 |
| 1·2위 강조 | 상위 2개 항목 시각적 강조 |

```
├── vote.html       # 투표 참여 페이지
├── result.html     # 결과 확인 페이지
└── PLAN.md         # 이 파일
```

### 데이터 저장 방식

**옵션 A — LocalStorage 방식 (단일 기기 데모용)**
- 별도 서버 불필요, 파일만으로 동작
- 단점: 관리자와 참여자가 같은 기기·브라우저여야 함 → 5G 환경 불가

**옵션 B — 외부 무료 DB 연동 방식 (추천)**
- [JSONBin.io](https://jsonbin.io) 또는 [Firebase Realtime DB](https://firebase.google.com) 무료 플랜 사용
- 인터넷 어디서나 접속 가능 → 5G 환경 지원
- HTML + JavaScript만으로 구현 가능 (서버 코드 불필요)

### 배포 방식

| 방법 | 설명 | 비용 |
|------|------|------|
| GitHub Pages | GitHub 저장소에 업로드 후 자동 배포 | 무료 |
| Netlify | 드래그앤드롭으로 즉시 배포 | 무료 |
| Vercel | GitHub 연동 자동 배포 | 무료 |

> 5G 환경 접속을 위해서는 반드시 위 방법 중 하나로 인터넷에 배포해야 합니다.

## 투표 흐름

```
[관리자]
  1. admin.html 열기
  2. 아이디어 항목 입력
  3. 투표 시작 버튼 클릭
  4. QR코드 화면에 표시

[참석자]
  5. QR코드 스캔 (스마트폰)
  6. vote.html 접속
  7. 아이디어 2개 선택
  8. 제출 버튼 클릭

[관리자 / 참석자]
  9. result.html에서 실시간 결과 확인
  10. 1·2위 아이디어 자동 표시
```

---

## 중복 투표 방지

- 투표 완료 후 브라우저 LocalStorage에 완료 플래그 저장
- 같은 기기·브라우저로 재접속 시 투표 불가 메시지 표시
- (추가 옵션) 세션 고유 ID 방식으로 서버 측에서도 검증

---

## 구현 우선순위

1. **1단계** — admin.html + vote.html + result.html 기본 구조 (LocalStorage)
2. **2단계** — JSONBin.io 연동으로 다기기 실시간 투표 구현
3. **3단계** — QR코드 생성, 결과 화면 UI 개선
4. **4단계** — 배포 및 5G 환경 테스트

---

## 참고 사항

- 1인당 선택 가능 아이디어 수: 기본 **2개** (관리자가 변경 가능)
- 결과 상위 2개 자동 강조 표시
- 모바일 최적화 UI (터치 친화적 카드 레이아웃)
